Capstone Design Templates

Greg Hislop

This module contains a set of templates used to help students create documents for team software projects in a capstone design sequence. The requirements specification and design specification are loosely based on the related IEEE standards (but scaled down considerably)

Learning Objectives

  • Application - Contribute as a team member to requirements specification, design, and implementation of a software product.

Pre-Requisites

  • Students must have at least general knowldege of the software development life cycle. Note that this should include an understanding of the phases involved, but need not be any particular model.
